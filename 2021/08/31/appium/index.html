appium环境搭建

node.js

下载地址：https://nodejs.org/en/download/releases/

命令：

C:\Users\admin>node -v

v6.11.3

C:\Users\admin>npm -v

3.10.10

注意：

如果显示‘npm’不是内部命令提示，则可以管理员省份运行cmd，如果还是失败，则需要检查一下环境变量是否配置，或者重新安装nodejs

Appium 安装

安装一：

1.安装最新版本

cnpm install -g appium

2.安装指定版本

​ cnpm install appium@1.7.2 -g

提示：

All packages installed (565 packages installed from npm registry, used 57s, speed 835.67kB/s, json 477(6.16MB), tarball 40.58MB)

[appium@1.8.0] link C:\Users\admin\AppData\Roaming\npm\appium@ -> C:\Users\admin\AppData\Roaming\npm\node_modules\appium\build\lib\main.js

安装二：

镜像设置

npm install -g cnpm –registry=https://registry.npm.taobao.org

执行完成命令看到如下提示则表示设置完成

提示：

C:\Users\admin> npm install -g cnpm –registry=https://registry.npm.taobao.org

appium配置

命令：

查看版本号：appium -v

启动服务：appium

提示：如果输入appium后显示：“appium不是内部或外部命令,也不是可运行的程序或批处理文件” 可以将appium安装的路径 如： “C:\Users\admin\AppData\Roaming\npm”配置到系统环境变量Path中

提出：ctrl+c 选择y 即可退出

安装Appium-desktop

下载地址：https://github.com/appium/appium-desktop/releases

安装成功之后输入命令 python –version

出现版本号

安装Appium-Python-Client

pip install Appium-Python-Client 进行安装。

在python下运行from appium import webdriver，无报错

Appium-Python-Client安装后的路径一般为：

{Python 安装路径}\Lib\site-packages\appium
