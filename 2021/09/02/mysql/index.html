一、在linux上面安装好mysql

准备：mysql(linux版本)

步骤：

\1. 传上去，在usr/local/src 在这里创建一个mysql的文件夹

\2. 检测一下自己的系统是否自带mysql

\1. 卸载掉自带的mysql版本

rpm -e –nodeps 版本号

在文件夹里面，解压mysql

\1. 安装server

指令：

安装依赖：yum -y install libaio.so.1 libgcc_s.so.1 libstdc++.so.6

libstdc++-4.4.7-4.el6.x86_64
yum update libstdc++-4.4.7-4.el6.x86_64

\1. 安装client(客户端)

如果安装client的时候，遇到error,则继续安装依赖

安装依赖：yum -y install libncurses.so.5 libtinfo.so.5

\1. 查询mysql的服务状态

服务端与客户端安装完毕

查询mysql的服务的状态

service mysql status

说明服务没有启动，接着启动服务

service mysql start

Mysql的服务启动成功

\1. 使用root来登陆mysql(忽视)

\1. 去找登陆密码：必须进入root目录里面来

root用户目录里面，去找一个隐藏文件

.mysql_secret

\1. 使用查看隐藏文件的指令 ll -a

\1. 使用vi编辑器去打开这个文件

vi .mysql_secret

复制下面这段原始密码–记得退出不要保存

lyv87tdvlw_0husj

\1. 使用mysql的root账户来登陆

mysql -uroot -p

\1. 输入刚刚复制的密码，然后登陆成功。 Lyv87TDvlw_0hUSJ(是我的原始密码)

\1. 登陆进来第一件事，改密码：

修改root密码：SET PASSWORD = PASSWORD(‘123456’);

密码修改成功，输入“exit;”退出

\1. 把mysql加入到系统的自启动

chkconfig –add mysql

\1. mysql自动启动

chkconfig mysql on

完成上述步骤，我们在linux上面安装好了mysql，mysql的服务也会随着linux系统启动而启动，但是它还不能支持远程访问。

\1. 开启远程访问(必须登陆进去)

A. 登陆到mysql当中去：

​ mysql -uroot -p

B. 设置mysql的远程访问的规则：

grant all privileges on . to ‘root’ @’%’ identified by ‘123456’;

​ Flush privileges;

\1. 打开防火墙规则，开启mysql的3306端口（必须退出mysql的登陆状态）

/sbin/iptables -I INPUT -p tcp –dport 3306 -j ACCEPT/etc/rc.d/init.d/iptables save/etc/init.d/iptables status

完成这一步，我们基本上可以去远程来访问我们mysql服务了。
